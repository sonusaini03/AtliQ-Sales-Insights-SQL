## AtliQ Sales Insights SQL Project
Welcome to my AtliQ Sales Insights repository! This project focuses on analyzing sales data using SQL to derive actionable business insights.

## Project Structure
As you can see, this project is organized into modular folders covering various SQL techniques and analytical reports:

> Foundation & Setup: Includes helpers, events, and triggers for database management.

> Database Views: Introduces views to simplify complex queries, including post-invoice and discount analysis.

> Sales Reporting: Contains scripts for Gross Sales reports (monthly and total).

> Performance Tuning: Optimization techniques implemented in folders like "Performance Improvement #1" and "#2".

> Advanced SQL: Covers Stored Procedures, Window Functions (OVER clause, ROW_NUMBER), and User-defined functions.

> Business Intelligence: Analysis focusing on Top Markets and Customers, as well as forecasting accuracy.

## Key Techniques Used
> Stored Procedures: Used for automating repetitive tasks like monthly growth and market badging.

> Window Functions: Applied to handle complex ranking and aggregation tasks efficiently.

> Data Optimization: Implemented performance improvements to ensure faster query execution.
